====== Building Player with CMake ======

===== Requirement =====
  * cmake-2.8 or greator
  * EasyRPG Reader
  * OpenAL
  * ffmpeg
  * PNG
  * SDL
  * SDL_image
  * SDL_ttf
  * ZLib

===== Build =====
  * Run "cmake ${PATH TO player/builds/cmake} && make".
  * If you want to build it with debug mode pass "-D CMAKE_BUILD_TYPE=Debug" to cmake.
