====== Readers requirements ======

===== Programs =====
  * C++ compiler (G++, MSVC...).
  * Make

===== Libraries =====

==== Required ====
  * libiconv for the character set detection and UTF-8 conversion.

==== Optional ====
  * Expat for XML reading support.
  * Google Test to perform some unit tests.

===== Optional programs =====
  * Doxygen to generate source code documentation.
  * Graphviz to generate graphs with the dot program used by Doxygen.
  * Python for the template source code regeneration.
